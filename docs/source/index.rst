Audiofab FV-1 VS Code Extension
================================

Welcome to the documentation for the **Audiofab FV-1 VS Code Extension** — a comprehensive development environment for creating audio effects for the `FV-1 DSP from Spin Semiconductor <http://www.spinsemi.com/products.html>`_ and the `Audiofab Easy Spin <https://audiofab.com/products/easy-spin>`_ guitar pedal.
